# Lab4-Machine-Learning

# Overview

This project performs basic data preprocessing steps on a dataset to prepare it for machine learning tasks. The workflow includes data cleaning, handling missing values, outlier removal, feature scaling, and dimensionality reduction.

# Dataset

The dataset is loaded from a CSV file (data.csv) and contains numerical features used for analysis.

# Steps Performed

Data inspection (structure, missing values, duplicates)
Data cleaning (handling ? values and converting data types)
Missing value treatment using mean imputation
Outlier detection and removal using IQR method
Feature scaling using Min-Max and Z-score normalization
Correlation analysis using heatmap
PCA applied for dimensionality reduction when correlation exists

# Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

# Output

Cleaned and processed dataset ready for machine learning analysis.
